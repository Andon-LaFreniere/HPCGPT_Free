# RecoveryGPT: Tailoring LLMs for Multimodal Athlete Recovery Planning
RecoveryGPT is a generative AI decision-support agent designed to deliver personalized, evidence-based recovery plans for elite athletes. Developed in collaboration with Ohio State Athletics, this system synthesizes multimodal inputs—including continuous biometric wearable data, subjective wellness surveys, and structured training schedules and grounds its recommendations in a curated peer-reviewed literature corpus using Retrieval-Augmented Generation (RAG).

This repository contains the conceptual framework, architectural workflows, and evaluation assets from the feasibility study: Tailoring LLMs for Multimodal Athlete Recovery Planning: Combining Wearable, Survey, and Training Schedule Data for Personalized Planning (The Ohio State University).
